MIDI Learn in Seq66

Seq66 uses three different types of MIDI Learn: Loops, Mutes, and Automation. The Loops and Mutes work by activating MIDI Learn and then having the user click the 32 (normally) buttons in order until enough have been assigned or all have been assigned. Automation works by clicking a user-interface item supported by Seq66's controls, and then clicking the device's button desired for that action.

The learn process starts with copying the current setup to a safe container that can be used to reset to the initial state. If desired, the safe container is copied to a working container; otherwise an empty control container is used. Any changes made can then be cleared, reset, or saved to a new 'ctrl' file.

The Learn Manager dialog provides user-interface elements for selecting the Learn mode, counting events that have been set to a device MIDI message, showing the currently set event, and allowing to clear, reset, start the process to log control event, and accept and save the controls.

The changes, if saved, are copied back to the performer object and take effect immediately.

Loops Learn

The basic process is to click the Start button, then, one after the other, click the desired buttons to toggle patterns 0, 1, 2, ... 32 (or higher if configured for larger grids). The process stops once the maximum number of grid slots are configured or Reset, Clear, Save, OK are clicked. Note that OK merely sets the "save" flag for the 'ctrl' file.

Mutes Learn

Similar to loops-learn, but applies to the buttons that select a mute-group.

Automation Learn

This mode cannot step through a serial list of controls. Instead, the user-interface item must be selected (i.e. clicked), then the desired device control must be clicked. Once clicked, the event is logged.
